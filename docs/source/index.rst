Izza is machine learning **Python toolkit**. It mainly implements tools for **assessing cluster tendency**. Cluster tendency is 
about discovering the existence of cluster's in multivariate data. (link to wikipedia)



**Note on Python 2.7**\ :


The python package is not available in python 2 and only tested on Linux for the moments.


**API Demo**\ :


.. code-block:: python

    from sklearn import datasets
    from izza.metrics import hopkins

    X = datasets.load_iris().data
    hopkins(X,150)
